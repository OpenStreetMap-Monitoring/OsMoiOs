# Настройка Firebase SDK
Для сборбки приложения необходимо установить модули Firebase SDK https://firebase.google.com/download/ios?authuser=0
Из  загруженного SDK скопировать в каталог Firebase  проекта:
Firebase.h
module.modulemap
FirebaseAnalytics
FirebaseMessaging
